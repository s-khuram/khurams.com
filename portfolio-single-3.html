Project Description

Built and optimized financial data pipelines supporting transaction processing, risk analysis, forecasting, and regulatory reporting. Migrated over 250 legacy ETL workflows into modular dbt and Airflow pipelines, reducing operational overhead by 70% and saving over $500K annually. Modernized real-time streaming architecture and reduced data latency from 10 minutes to under 1 minute with Kafka and Kubernetes. Built CI/CD pipelines for data engineering workflows using Jenkins, GitHub Actions, and Docker, enabling automated testing, validation, and multi-environment deployment of ETL pipelines. Collaborated with data science teams to operationalize ML pipelines, reducing model deployment times by 50%. Developed a self-service analytics framework that empowered more than 100 business users with real-time data access. Improved query performance by 35% through model optimization and indexing strategies. Ensured zero data-loss tolerance through fault-tolerant ingestion architecture. Orchestrated multi-cloud cost optimization strategies, reducing infrastructure spend by 22%. Led Data Mesh adoption, reducing cross-team data access bottlenecks by 40%. Worked in Agile/Scrum environments using Jira and Confluence to manage data platform delivery and cross-team coordination. Owned production support, on-call rotations, and incident resolution for critical data pipelines, ensuring SLA compliance and system reliability.
