MediTech

Project Date 03/2021 - Present
Role Senior Data Engineer
Client MediTech
Visit https://ehr.meditech.com/
Project Description

Designed and operated large-scale healthcare data platforms across AWS, Azure, and GCP supporting 15+ business domains in regulated environments. Engineered HIPAA-compliant data pipelines for healthcare datasets including EHR, claims, eligibility, and clinical events, ensuring PHI-safe ingestion and processing. Built high-throughput streaming pipelines using Kafka and Spark, processing 5M+ events/hour with sub-3-second latency for clinical, claims, and financial data. Designed and optimized Snowflake data warehouses, implementing CDC pipelines (Snowpipe, Streams, Tasks) that reduced annual query costs by $1.2M. Developed real-time claims and clinical event processing pipelines with sub-minute latency to support fraud detection and utilization monitoring. Implemented PHI protection frameworks using encryption, masking, RBAC/ABAC, and audit logging to meet HIPAA, GDPR, and CCPA requirements. Engineered a hybrid cloud and on-prem data platform designed to handle more than 10TB of daily throughput using Spark, Kafka, and Databricks. Built FHIR- and HL7-aligned data models enabling downstream analytics, reporting, and ML consumption across clinical and operational teams. Built resilient ETL pipelines using Delta Lake, Airflow, and Python to process over 5TB+ of data daily. Launched a self-service analytics platform enabling 300+ users to access near-real-time insights without engineering dependency. Implemented Data Observability solutions such as Monte Carlo and Great Expectations, reducing data quality issues by 60%. Mentored six engineers and improved CI/CD deployment success rates from 70% to 98%. Created pipeline documentation, architecture diagrams, and operational runbooks to improve maintainability and team onboarding. Partnered with stakeholders, product teams, and data scientists to translate business requirements into reliable analytics datasets and data platform solutions.
